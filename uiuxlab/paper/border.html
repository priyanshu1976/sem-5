CSS Borders

Complete guide to styling borders with CSS

Introduction to CSS Borders

The border property is used to create a border around an element, such as a div, image, or text. It allows you to customize the appearance of the border, including its color, width, and style.

Types of Border Properties

  • border-style: Specifies whether a border should be solid, dashed line, double line, or one of the other possible values
  • border-width: Specifies the width of a border
  • border-color: Specifies the color of a border

Border Style Property

The border-style property is used to specify the type of border style for an element.

No border.

Hidden

Hidden border. 
border-style: hidden;

Dotted border.

Dashed

Dashed border. 
border-style: dashed;

Solid

Solid border. 
border-style: solid;

Double

Double border. 
border-style: double;

Groove

Groove border. 
border-style: groove;

Ridge

Ridge border. 
border-style: ridge;

Inset

Inset border. 
border-style: inset;

Outset

Outset border. 
border-style: outset;

Border Style For Individual Sides

The border-style property can be specified for each side individually.

Example Code

p {
    border-top-style: dotted;
    border-right-style: solid;
    border-bottom-style: dashed;
    border-left-style: double;
    padding: 2em;
    width: 150px;
}

Border Width Property

The border-width property is used to specify the thickness of a border around an element. It can have values like thin, medium, thick or any length (in px, or em).

Thin

Thin width. 
border-width: thin;

Medium

Medium width. 
border-width: medium;

Thick

Thick width. 
border-width: thick;

Specific Value (10px)

Border width: 10px. 
border-width: 10px;

Different Widths on Each Side

p {
    border-style: solid;
    border-top-width: thin;
    border-right-width: thick;
    border-bottom-width: medium;
    border-left-width: 10px;
}
Different border widths on all sides.

Border Color Property

The border-color property is used to set the color of a border.

Single Color

p {
    border-style: solid;
    border-color: blue;
}
Border color blue.

Different Colors on Each Side

p {
    border-style: solid;
    border-top-color: red;
    border-right-color: #0000ff;
    border-bottom-color: rgb(100,123,111);
    border-left-color: rgba(50,123,111,0.4);
}
Different border color on all sides.

Border Shorthand Property

In CSS, we can use the border property to specify style, width and color of a border in one line.

Syntax

h2 {
    border: 4px dotted green;
}

The above code will add a green, dotted and 4px thick border on all the four sides of the h2 element.

Border Radius Property

The CSS border-radius property is used to specify the roundness of border edges. The value for this property can be in length (px, em) or percentages. A border radius of 50% indicates a complete circle.

Example Code

#round {
    border-radius: 25px;
}

#circle {
    border-radius: 50%;
}

Rounded Corners (25px)

border-radius: 25px;

Circle (50%)

border-radius: 50%;

CSS Image As Border

CSS also allows you to set images as borders for elements like div, span, body, p etc. using the border-image property. Declare the border-style before providing an image source, else no image will be displayed as the border.

Example Code

div {
    background-color: #ffffff;
    border: 20px solid;
    border-image: url(images.jpg) 40;
    padding: 20px;
}
