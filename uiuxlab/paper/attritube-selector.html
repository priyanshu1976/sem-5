Theory & Interactive Examples

Attribute selectors let you style elements based on their attributes (like type, href, alt, src, etc). This is super useful for targeting elements without adding extra classes or IDs, keeping your CSS clean, organized, and scalable!

Types of Attribute Selectors:

  1. [attr] ‚Äì Has Attribute
  2. [attr=value] ‚Äì Exact Match
  3. [attr~=value] ‚Äì Contains Word
  4. [attr^=value] ‚Äì Starts With
  5. [attr$=value] ‚Äì Ends With
  6. [attr*=value] ‚Äì Contains Anywhere
  7. Case-Insensitive Matching (i or I)
  8. Case-Sensitive Matching (s or S)

1. [attr] ‚Äì Has Attribute

Styles elements that have a specific attribute (any value).
CSS:
[disabled] { background-color: #ffd6d6; border: 2px solid red; }
HTML:
<button disabled>Disabled Button</button>
<button>Enabled Button</button>
‚úÖStyles all <button disabled>

2. [attr=value] ‚Äì Exact Match

Selects elements where attribute value matches exactly.
CSS:
input[type="text"] { background-color: #f0f8ff; }
HTML:
<input type="text" placeholder="Styled!">
<input type="password" placeholder="Not styled">
‚úÖOnly <input type="text"> gets background
‚ùå<input type="password"> not affected

3. [attr~=value] ‚Äì Contains Word

Selects elements whose attribute value is a space-separated list of words, and one matches exactly.
CSS:
div[class~="featured"] { border: 2px solid green; }
HTML:
<div class="featured card">...</div>
<div class="featuredbox">...</div>
I have class="featured card" ‚Üí I am styled
I have class="featuredbox" ‚Üí Not styled
‚úÖ<div class="featured card"> is selected
‚ùå<div class="featuredbox"> is NOT selected

4. [attr^=value] ‚Äì Starts With

Selects elements whose attribute value starts with the given value.
CSS:
a[href^="https"] { color: green; }
HTML:
<a href="https://...">...</a>
<a href="http://...">...</a>
Starts with https ‚Üí Styled
Starts with http (no s) ‚Üí Not Styled
‚úÖ<a href="https://..."> is green
‚ùå<a href="http://..."> not selected

5. [attr$=value] ‚Äì Ends With

Selects elements whose attribute value ends with the given value.
CSS:
img[src$=".png"] { border: 2px solid red; }
HTML:
<img src="logo.png">
<img src="logo.jpg">
‚úÖ<img src="logo.png"> has border
‚ùå<img src="logo.jpg"> not affected

6. [attr*=value] ‚Äì Contains Anywhere

Selects elements whose attribute value contains the value anywhere.
CSS:
a[href*="google"] { background-color: yellow; }
HTML:
<a href="https://www.google.com">...</a>
<a href="https://www.bing.com">...</a>
Contains google ‚Üí Styled
Does not contain google ‚Üí Not Styled
‚úÖ<a href="https://www.google.com"> is styled
‚ùå<a href="https://www.bing.com"> not affected

7. Case-Insensitive Matching (i or I)

Add i to make the match case-insensitive.
CSS:
a[href="EXAMPLE" i] { color: purple; }
HTML:
<a href="example">...</a>
<a href="EXAMPLE">...</a>
<a href="ExAmPlE">...</a>
Matches (example)
Matches (EXAMPLE)
Matches (ExAmPlE)
‚úÖMatches <a href="example">, <a href="EXAMPLE">, <a href="ExAmPlE">
Another Example (with data-info):
p[data-info="hello" i] { color: purple; font-style: italic; }

Matches hello (case-insensitive)

Matches hello (case-insensitive)

8. Case-Sensitive Matching (s or S)

Add s to force case-sensitive match (default in most browsers).
CSS:
a[href="Example" s] { color: red; }
HTML:
<a href="Example">...</a>
<a href="example">...</a>
<a href="EXAMPLE">...</a>
Matches (Example)
Does NOT match (example)
Does NOT match (EXAMPLE)
‚úÖOnly matches <a href="Example">
‚ùåWon‚Äôt match <a href="example"> or <a href="EXAMPLE">
Another Example (with data-info):
p[data-info="CaseSensitive" s] { color: red; text-decoration: underline; }

Matches EXACT case

Does NOT match (different case)
