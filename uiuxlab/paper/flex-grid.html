Quick Theory

display decides how an element participates in layout (block/inline/inline-block/flex/grid/none). Block consumes full width and starts on new line; inline flows inside lines.

Flexbox is one-dimensional (main + cross axis). Use for linear alignment, centering, nav bars, responsive rows/columns.

Grid is two-dimensional—rows + columns—useful for full page layouts and complex placements.

display — theory & examples

How elements are rendered. Default depends on element type.

Common values

  • block — occupies full available width; starts on new line (e.g. <div>).
  • inline — flows within a line; width = content (e.g. <span>).
  • inline-block — like inline but supports width/height.
  • none — element removed from layout/flow.
  • flex — creates a flex container; children are flex items.
  • grid — creates a grid container; children are grid items.

Example — inline vs block vs inline-block: 

<div style="display:block">Block</div>
<span style="display:inline">Inline text</span>
<div style="display:inline-block; width:120px; height:40px">Inline-block</div>
Try toggling these examples in your own pages — block breaks line, inline does not, inline-block keeps dimension.

Flexbox — parent & item properties

One-dimensional layout along a main axis and cross axis. Container properties control flow; item properties control sizing and alignment.
Per-item visual controls:
Live preview — change controls to update the flex container

/* container */
.container {
  display: flex;
  flex-direction: row;
  flex-wrap: nowrap;
  justify-content: flex-start;
  align-items: stretch;
  align-content: stretch;
}

/* items */
.item { flex: 0 1 auto; align-self: auto; }
Item properties — flex-grow, flex-shrink, flex-basis, shorthand flex, and align-self.
E.g. .item { flex: 1 0 150px; } — grow to fill, don't shrink, base 150px.
Example code (center horizontally & vertically): 
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
</div>

.container{ display:flex; justify-content:center; align-items:center; height:260px; }
.item{ padding:10px; background:#064; color:white; border-radius:6px; }

CSS Grid — parent & item properties

Two-dimensional layout — rows and columns. Use `grid-template-*`, `gap`, `grid-auto-flow`, and place items using `grid-column`, `grid-row`, or `grid-area`.
Or use shorthand: grid-column: 2 / 4;

  display: grid;
  grid-template-columns: 1fr 1fr 1fr;

.item { padding:10px; background:#063544; color:#dff3ff; }
Examples of placement 
/* place an item spanning columns 1 to 3 */
.item1 { grid-column: 1 / 3; grid-row: 1 / 2; }

/* shorthand grid-area: row-start / col-start / row-end / col-end */
.item2 { grid-area: 1 / 3 / 3 / 4; }
