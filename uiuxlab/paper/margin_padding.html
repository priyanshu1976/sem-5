CSS Margin and Padding

Complete guide to spacing elements with CSS

Understanding the Box Model

In CSS, every element is represented as a rectangular box. The box model describes how the content, padding, border, and margin of an element work together to create its visual appearance.

Content
Margin Border Padding Content

Key Components:

  • Content: The actual content of the box, where text and images appear
  • Padding: Clears an area around the content (inside the border)
  • Border: A border that goes around the padding and content
  • Margin: Clears an area outside the border (transparent)

CSS Margin Properties

The margin property controls the gap between boxes. Its value is commonly given in pixels, although you may also use percentages or ems.

Margin Properties

Property Description
margin A shorthand property for setting all the margin properties in one declaration
margin-bottom Sets the bottom margin of an element
margin-left Sets the left margin of an element
margin-right Sets the right margin of an element
margin-top Sets the top margin of an element

Margin Values

All the margin properties can have the following values:

  • auto - the browser calculates the margin
  • length - specifies a margin in px, pt, cm, etc.
  • % - specifies a margin in % of the width of the containing element
  • inherit - specifies that the margin should be inherited from the parent element

Different Ways to Apply Margin

There are four ways to provide value to the CSS margin property:

Single Value

Apply the same margin to all four sides

margin: 25px;
25px

All four margins are 25px

Two Values

First value: top and bottom
Second value: right and left

margin: 25px 50px;
25px 50px

Top and bottom: 25px
Right and left: 50px

Three Values

First: top, Second: right and left, Third: bottom

margin: 25px 50px 75px;
25px 50px 75px

Top: 25px, Right/Left: 50px, Bottom: 75px

Four Values

Top, Right, Bottom, Left (clockwise)

margin: 25px 50px 75px 100px;
25px 50px 75px 100px

Top: 25px, Right: 50px, Bottom: 75px, Left: 100px

Margin with Mixed Units

CSS does not restrict usage of multiple units for specifying the values of margin, while specifying in shorthand property.

Example

h2 {
    margin: 20px 4ex 0.5in 3em;
}
This element has mixed margin units: 20px top, 4ex right, 0.5in bottom, and 3em left.

Negative Margins

A margin with negative value indicates the elements overlap with each other.

Negative Margin Example

.box1 {
    margin-right: -20px;
}

Box 2

Notice how Box 1 overlaps Box 2 due to the negative right margin.

Auto Value for Margins

You can set the margin property to auto to horizontally center the element within its container.

Centering with Auto Margins

div {
    width: 300px;
    margin: auto;
    background-color: lightblue;
    border: 2px solid red;
}
This element is centered using margin: auto;

The element takes up the specified width, and the remaining space is split equally between the left and right margins.

CSS Padding Properties

The padding property controls the space between an element's content and its border. Like margin, it can be specified using various units.

Padding Properties

Property Description
padding A shorthand property for setting all the padding properties in one declaration
padding-bottom Sets the bottom padding of an element
padding-left Sets the left padding of an element
padding-right Sets the right padding of an element
padding-top Sets the top padding of an element

Padding Values

All the padding properties can have the following values:

  • length - specifies a padding in px, pt, cm, etc.
  • % - specifies a padding in % of the width of the containing element
  • inherit - specifies that the padding should be inherited from the parent element

Note: Unlike margin, padding does not accept negative values or the auto value.

Different Ways to Apply Padding

Like margin, the padding property can accept one to four values:

Single Value

Apply the same padding to all four sides

padding: 20px;
20px

All four paddings are 20px

Two Values

First value: top and bottom
Second value: right and left

padding: 20px 40px;
20px 40px

Top and bottom: 20px
Right and left: 40px

Three Values

First: top, Second: right and left, Third: bottom

padding: 20px 40px 60px;
20px 40px 60px

Top: 20px, Right/Left: 40px, Bottom: 60px

Four Values

Top, Right, Bottom, Left (clockwise)

padding: 20px 40px 60px 80px;
20px 40px 60px 80px

Top: 20px, Right: 40px, Bottom: 60px, Left: 80px

Margin vs Padding: When to Use Which

Use Margin When:

  • You want to create space between elements
  • You want to center an element horizontally
  • You need negative spacing (overlapping elements)
  • You want to adjust the position of an element relative to its neighbors

Use Padding When:

  • You want to create space between the content and border of an element
  • You want to increase the clickable area of a button or link
  • You want to control the background area around the content
  • You need to push content away from the borders inside a container
