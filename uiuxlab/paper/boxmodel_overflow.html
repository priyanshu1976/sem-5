The CSS Box Model describes how every HTML element is a rectangular box.
It consists of content, padding, border, and margin layers:

  • Content: The actual text or image inside the box.
  • Padding: Space between the content and the border (inside).
  • Border: The edge around the padding.
  • Margin: Space outside the border, separating elements.
