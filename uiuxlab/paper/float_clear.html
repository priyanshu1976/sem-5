CSS Float and Clear

Complete guide to floating elements and clearing floats

Introduction to CSS Float

The CSS float property specifies how an element should float. A floated element is taken out of the normal flow and placed along the left or right side of its container, allowing text and inline elements to wrap around it.

Float Properties

Property Description
float: left; The element floats to the left of its container
float: right; The element floats to the right of its container
float: none; The element does not float (default)
float: inherit; The element inherits the float value of its parent

Float Values and Examples

Float: Left

Float Left

This text wraps around the element that is floated to the left. The float property positions the element to the left side of its container, and other content flows around it on the right side. 

.float-left {
    float: left;
}

Float: Right

Float Right

This text wraps around the element that is floated to the right. The float property positions the element to the right side of its container, and other content flows around it on the left side. 

.float-right {
    float: right;
}

Float: None

Float None

This element is not floated, so it appears in the normal flow of the document. Text and other elements do not wrap around it but instead appear above and below it as part of the normal document flow. 

.float-none {
    float: none;
}

The Clear Property

The clear property controls the behavior of elements next to floated elements. It specifies what should happen with the element that is next to a floated element.

Clear Properties

Property Description
clear: left; The element is moved down to clear left floats
clear: right; The element is moved down to clear right floats
clear: both; The element is moved down to clear both left and right floats
clear: none; The element is not moved down to clear any floats (default)
clear: inherit; The element inherits the clear value from its parent

Creating Layouts with Float

Before Flexbox and Grid became widely supported, float was the primary method for creating multi-column layouts.

Two-Column Layout

Header
Sidebar (float: left)
Main Content (float: right)
Footer (clear: both)
.sidebar {
    float: left;
    width: 25%;
}

.content {
    float: right;
    width: 70%;
}

.footer {
    clear: both;
}

Three-Column Layout

Column 1 (float: left)
Column 2 (float: left)
Column 3 (float: left)
This clears the floats
.column {
    float: left;
    width: 30%;
    margin: 0 1.5%;
}

The Clearfix Hack

When elements are floated, their parent container often collapses (has a height of 0). The clearfix hack solves this problem.

Without Clearfix

Float Left
Float Left

Parent container has collapsed because it only contains floated elements.

With Clearfix

Float Left
Float Left

Parent container maintains its height thanks to the clearfix.

Clearfix Code

.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

Apply the clearfix class to any container that has floated children to prevent collapse.

Float in Modern CSS

While float is still useful for text wrapping around images, modern layout techniques like Flexbox and CSS Grid are generally preferred for creating page layouts.

When to Use Float

  • Wrapping text around images
  • Creating simple side-by-side content
  • Legacy browser support
  • When you need content to flow around an element

When to Use Modern Layout Techniques

  • Complex page layouts
  • Responsive designs that need to reflow content
  • Vertical centering
  • When you need equal height columns
  • Most new projects
